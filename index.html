All 12 cases explained

Sentence case

Capitalizes the first letter of each sentence. Great for fixing accidental caps-lock text.

lower case

Converts every letter to lowercase. Useful for normalizing input or casual writing styles.

UPPER CASE

Makes all letters capitals. Perfect for headings, shouts, and acronyms.

Capitalized Case

First letter of every word is capitalized. Common for names and titles.

aLtErNaTiNg cAsE

Alternates between lower and upper on each character. Fun and mocking-spongebob vibes.

Title Case

Capitalizes important words, skipping articles and prepositions. Ideal for article headlines.

iNVERSE cASE

Flips the case of every letter — uppercase becomes lowercase and vice versa.

camelCase

No spaces, each word starts with a capital except the first. Standard in JavaScript variables.

PascalCase

Like camelCase but the first word is also capitalized. Used for class names in most languages.

snake_case

Words joined by underscores in lowercase. Popular in Python and database column names.

kebab-case

Words joined by hyphens in lowercase. Standard for URLs, CSS class names, and file names.

CONSTANT_CASE

All caps with underscores. Used for constants and environment variables in most languages.
